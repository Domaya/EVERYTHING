What is SVG?

SVG stands for Scalable Vector Graphics
SVG is used to define graphics for the Web
SVG is a W3C recommendation

The HTML svg Element

The Html svg element is a container for SVG graphics.
SVG has several methods for drawing paths, boxes, circles, text, and graphic images.
HTML svg 요소는 svg 그래픽을 위한 컨테이너일 뿐이다.
SVG는 path,박스,원,문자,그래픽 이미지를 그리기 위한 많은 메소드를 가지고 있다.

SVG와 Canvas의 차이점?

SVG는 2d그래픽을 xml에서 그리기 위한 언어이다.
캔버스는 2D 그래픽을 자바스크립트로 그려낸다.

SVG는 xml 기반으로, 모든 요소는 SVG DOM에서 사용 가능하다. 사용자는 자바스크립트 이벤트 핸들러로 이러한 요소에 접근 가능하다.
SVG에서 각각의 그려진 모양은 객체로 기억된다. 만약 SVG 객체의 요소가 변경된다면, 브라우저는 자동으로 이러한 요소를 리렌더링한다.
캔버스는 픽셀과 픽셀로 렌더링 되어 있다. 캔버스에서 그래픽이 한 번 그려지면 이는 브라우저에서 잊힌다. 만약 그려진 모양이 변경되야 한다면, 모든 것이 다시 그려져야 한다.
